## 下载方法

### 方法1：使用下载脚本
```bash
# 列出可用资源
python3 download_dict.py list

# 下载词典
python3 download_dict.py <URL> [文件名]
```

### 方法2：手动下载
1. 从GitHub或其他资源下载词典文件
2. 将文件保存到 `data/dicts/` 目录
3. 确保文件格式：每行一个词条，UTF-8编码

### 方法3：从现有项目提取
如果你有imewlconverter项目，可以：
```bash
# 从imewlconverter提取词库
cp imewlconverter/参考/8万精准超小词库.txt data/dicts/chinese_common.txt
```

## 词典文件要求

- 格式：每行一个词条
- 编码：UTF-8
- 扩展名：`.txt`
- 支持 `#` 开头的注释行
- 支持制表符分隔（只取第一列）

## 使用

下载词典后，过滤脚本会自动从 `data/dicts/` 目录查找并使用词典文件。
